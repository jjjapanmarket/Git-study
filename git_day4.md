# Git学習day4

## ブランチとは
- 並行して複数機能を開発するためにあるのがブランチ
- コミットしたらブランチが指すコミットファイルが変わる

## HEAD
- 現在作業中のブランチへのポインタ。

## ブランチを新規追加する
- git branch<ブランチ名>
### ブランチの一覧を表示する
- git branch (-a 全てのブランチを表示する)

## ブランチを切り替える
- git checkout<既存ブランチ名>
### ブランチを新規作成して切り替える
- git checkout -b<新ブランチ名>

##　マージとは
- 他の人の変更内容を取り込む作業のこと
- git merge<ブランチ名>
